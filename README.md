#  Smart City Route Optimization System

##  Overview
This project implements multiple **Data Structures and Algorithms (DSA)** concepts in a real-world scenario of a smart city road network.

It includes:
- Graph Algorithms
- Dynamic Programming
- Backtracking
- Algorithm Comparison System

---

##  Features

###  Greedy Algorithms
- Dijkstra’s Algorithm
- Prim’s Algorithm
- Activity Selection

###  Dynamic Programming
- Floyd Warshall
- Bellman Ford
- Travelling Salesman Problem (TSP)

###  Backtracking
- N-Queens Problem
- Subset Sum Problem

###  Additional Features
- Graph type detection (Sparse/Moderate/Dense)
- Algorithm recommendation system
- Time complexity comparison
- Smart City visual map

---

##  Concepts Used
- Graph Theory
- Dynamic Programming
- Backtracking
- Greedy Algorithms
